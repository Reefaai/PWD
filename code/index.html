Membuat Project UAS Mata Kuliah Algoritma dan Pemograman Lanjut

Project UAS adalah sebuah kegiatan proyek atau tugas akhir yang dilakukan oleh siswa atau mahasiswa sebagai bentuk penilaian akhir dalam suatu mata kuliah. Tujuannya adalah untuk mengevaluasi sejauh mana pemahaman dan penerapan materi yang diajarkan selama perkuliahan melalui pembuatan produk atau karya nyata, bukan hanya melalui ujian tertulis.

Dalam proyek mata kuliah Algoritma dan Pemrograman Lanjut, setiap mahasiswa ditugaskan untuk membuat sebuah program pendataan atau pencatatan berbasis Graphical User Interface (GUI). Program ini dikembangkan menggunakan library Tkinter, dengan dukungan basis data yang dapat dipilih antara SQL maupun file Excel (CSV) sebagai media penyimpanan.

Bagi saya, tugas ini menjadi tantangan baru karena sebelumnya saya belum pernah membuat program serupa. Melalui proses pengerjaan proyek ini, saya banyak belajar mengenai bagaimana merancang dan menyusun struktur basis data, mulai dari menentukan kolom yang tepat hingga memilih tipe data yang sesuai. Selain itu, saya juga berlatih menyusun antarmuka pengguna (User Interface) agar program dapat lebih mudah digunakan oleh pengguna nantinya.
